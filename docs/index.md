## 站点介绍

这个站点用于系统整理 JavaScript 引擎学习与安全研究相关内容，强调从基础概念、源码阅读、调试实践到漏洞分析的连续学习路径。

当前内容主要围绕四条主线展开：

- **JSC**：理解执行层、优化机制、栈布局与调试方法
- **V8**：整理 JIT、TurboFan 与漏洞分析相关笔记
- **ChakraCore**：归纳执行流程、IR、调试参数与 Root Cause 分析
- **VSCode 配置**：整理 clangd、源码索引与调试环境搭建流程
